## [0.1.0] - 2026-05-17

### Added

- **`gtrans`** — Google Translate CLI with clipboard support and chunked translation for long texts.
- **`charcount`** — Character counter for text or files with an optional maximum-length check.
- **`imgmin`** — Visually-lossless image compression toolkit supporting JPEG, PNG, and WebP via sharp and mozjpeg.
- **`resize-image`** — Resize JPG, PNG, and WebP images to a target dimension via ImageMagick.
- **`towebp`** — Batch-convert PNG/JPG/JPEG images to WebP via cwebp.
- **`html2md`** — Convert HTML files to Markdown via pandoc.
